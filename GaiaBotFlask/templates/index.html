GaiaBot Controls

Welcome

This is a simple web interface to control the GaiaBot. You can turn it on and off using the buttons below.

The GaiaBot can be powered on using the "Start" button.

GaiaBot will cease collection, deposit its basket and return to the charging port after clicking the "Return" button.

If you want to stop and power off the bot immediately, you can use the "Off" button.
